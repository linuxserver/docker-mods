# Security Policy

## Supported Versions

Only the latest major and minor version are supported.

| Version | Supported          |
| ------- | ------------------ |
| 2.16.x  | :heavy_check_mark: |
| < 2.16  | :x:                |

## Reporting a Vulnerability

Vulnerability reports or security concerns may be submitted as a new issue in GitHub, or emailed to:
